1R: Data Binding em angular são mecanismos que servem para passar informações entre diferentes componentes da aplicação
2R: Sempre que um componente ultiliza outros componentes é estabelecido uma relação de pai para filho, o componente que usa é o pai e o componente usado é o filho
3R: Interpolation, Property Binding e Event Binding
4R: Event Binding
5R: Você executa o metodo emit() quando você quer passar uma informação de um componente filho para o pai, para isso, emitimos um evento usando emit() no filho e capturamos e manipulamos esse evento no componente pai.
6R: Um manipulador de evento é o que usamos no componente pai para capturar um evento (vindo do filho) e executar um código que vai dizer o que deve ser feito quando esse evento acontece
7R: @Input
8R: interpolação em Angular é quando você exibe na view de um componente um valor proveniente de sua lógica
9R: Você usa Property Binding
10R: Você usa Event Binding
11R: EventEmitter
12R: splice em um array serve para remover ou substituir itens de um determinado array.
